Buscar gasolineras abiertas

En este apartado del manual de usuario, se le va a explicar cómo funciona el Pop Up de filtros desarrollado en la aplicación y en especial el funcionamiento del filtro de abierto, mediante el cual podrás buscar gasolineras que se encuentren abiertas en ese preciso momento.

Pop Up filtros 🔎

Debe tener en cuenta que la lista de gasolineras que aparece en la interfaz principal, previamente explicada, muestra todas las gasolineras existentes en España, es por esto, por lo que se han desarrollado una serie de filtros de búsqueda para que le sea mucho más sencillo encontrar una gasolinera deseada.

A partir de este Pop Up, usted podrá aplicar unos filtros de búsqueda en el caso de que desee encontrar una o unas determinadas gasolineras para su uso. En este Pop Up podrá hacer uso de un desplegable mediante el cual se le mostrarán todas las provincias de España y a continuación otro desplegable en el que podrá ver todos los municipios de la provincia previamente seleccionada. Por último, encontramos una check box denominada Abierto con la que podrá filtrar aquellas gasolineras que se encuentren abiertas en ese preciso momento.

En el siguiente ejemplo podrá ver el uso de estos filtros, aunque en el siguiente apartado solo le explicaremos cómo hacer uso del filtro abierto. Si usted desea comprender el uso de los filtros de provincia y municipio, vuelva al índice o pinche aquí.

¿Desea buscar gasolineras que se encuentren abiertas? 🤔

Para facilitar la búsqueda de gasolineras abiertas, la aplicación ofrece un filtro de abierto. Al pinchar en la check box Abierto, la lista de gasolineras se actualizará de forma que solo se muestren aquellas gasolineras que actualmente están abiertas.

Este sistema de filtros hace que la búsqueda de gasolineras sea una tarea sencilla y efectiva.
